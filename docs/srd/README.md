## How to Use

- **For class features at a specific level:** Search `02 classes.md` for the class name, then check the level progression table
- **For spell details:** Search `08 spellcasting.md` for the spell name
- **For combat rules:** Read `07 combat.md` for actions, movement, attacks
- **For conditions:** Read `12 conditions.md` (short file, read whole thing)
- **For monster stats:** Search `11 monsters.md` for the creature name
- **For ability checks / skills:** Read `06 mechanics.md`
